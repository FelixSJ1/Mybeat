# Mybeat
Uma aplicação de gerenciamento de albums e músicas. Isso é, permite a manipulação CRUD de dados de música e o usuário pode registrar suas músicas, avaliá-las com notas.


             	Rede social de avaliação de músicas


Sprint 1: Implementação das manipulações de músicas.

Valor do sprint: permitir que o usuário acesse a plataforma, para que ele acesse a lista de músicas e faça adições, exclusões e atualizações.

Funcionalidades:

1-Criar uma página para os principais processos de manipulação de dados.
2-Adicionar músicas 
3-Excluir músicas
4-Editar os dados das músicas
5- Mostrar os dados já presentes


Sprint 2: Criação da divisão de papéis entre administrador e usuário

Valor do sprint: Habilitar a criação de usuários e administradores para um teste mais efetivo das próximas funções

Funcionalidades:

Cadastrar usuários através de uma página
Cadastrar administradores através de uma página a parte
Dividir papéis eficientes 
Visualizar ambas as páginas



User Stories

Adicionar, excluir, editar, mostrar músicas:

Como administrador, quero poder adicionar álbuns, músicas e EP’s e manipular esses dados livremente.


Mostrar as músicas:
Como usuário, quero poder ver as músicas já presentes no catálogo para descobrir mais sobre meus gostos musicais.



criação de usuários:

Como um Visitante, eu quero criar uma conta usando meu e-mail e senha, para que eu possa ter um perfil com nickname e começar a registrar as músicas que ouço.

Para o login e autenticação:

Como um usuário cadastrado, eu quero fazer login na minha conta, para que eu possa acessar meu perfil e minhas funcionalidades personalizadas.

Para a diferenciação de papéis:

Como Administrador, eu quero ter acesso a um painel de gerenciamento que não é visível para usuários comuns, para que eu possa executar minhas tarefas de edição do catálogo de forma segura.
